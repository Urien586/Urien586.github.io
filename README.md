# EYG Grafik & Matbaa

EYG Grafik & Matbaa için hazırlanmış statik tanıtım sitesi.

## İçerik

- Grafik tasarım ve matbaa hizmetleri
- Kartvizit, broşür, magnet, afiş, branda, etiket ve sticker ürünleri
- Örnek iş görselleri
- Instagram ve WhatsApp yönlendirmeleri
- Açık/koyu tema seçimi

## GitHub Pages Yayınlama

1. Bu projeyi GitHub reposuna yükleyin.
2. GitHub'da `Settings > Pages` bölümüne girin.
3. `Deploy from a branch` seçin.
4. Branch olarak `main`, klasör olarak `/root` seçin.
5. Kaydedin.

Site ana dosyası `index.html` olduğu için ek kurulum gerektirmez.
